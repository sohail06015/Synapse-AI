# Synapse AI

Synapse AI is a full-stack AI SaaS platform that provides multiple AI-powered tools through a modern and intuitive interface. Users can chat with Google's Gemini AI, generate blog titles and articles, create AI-generated images, review resumes, remove image backgrounds, and manage AI conversations with persistent chat history.

# Features

## Authentication

- Secure JWT Authentication
- Protected Routes
- User Session Management

## AI Chat

- Chat with Google Gemini AI
- Multiple chat sessions
- Persistent chat history
- Context-aware conversations
- Markdown rendering
- Syntax-highlighted code blocks
- Copy code functionality
- Typing/loading animation
- Auto-scroll to latest message

## AI Article Writer

- Generate high-quality articles on any topic
- Multiple writing styles
- Well-structured AI-generated content

## AI Blog Title Generator

- Generate engaging blog titles
- SEO-friendly title suggestions
- Multiple title variations

## AI Image Generator

- Generate images from text prompts
- High-quality AI-generated images
- Download generated images

## Resume Reviewer

- Upload resumes for AI analysis
- ATS-friendly recommendations
- Resume improvement suggestions

## Background Remover

- Remove image backgrounds using AI
- High-quality processed images
- Fast image processing

## Dashboard

- Access multiple AI tools from one platform
- Clean and responsive interface
- Easy navigation between AI utilities
